1.建立Git本地环境
 git init 

2.链接github
git remote add origin https://github.com/Eire-deerlet/***

3.添加代码到本地
git add .

4.添加注释
git commit -m "代码注释"

5.上传github
git push origin master

6.退代码版本

	1 项目右键选择然后点击 show log

	2 选中需要回退的代码版本

	3.右键  选择“ Reset "master to this ”

	4. 类型选择 “hard”，点击确定
